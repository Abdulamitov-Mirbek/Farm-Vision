# Установка и запуск Agro Suite

## 📋 Предварительные требования

### Системные требования
- Node.js 18 или выше
- MongoDB 4.4 или выше
- npm 9 или выше
- Git

### Рекомендуемые инструменты
- VS Code или другая IDE
- Postman или Insomnia для тестирования API
- MongoDB Compass для управления базой данных

## 🚀 Быстрый старт

git clone https://github.com/yourusername/agro-suite.git
